Presets 

Blank Documents Presets are predefined dimensions and settings. 

Presets make designing for specific paper size or specific device form factors. 

For example, you can use a preset to quickly start designing for the A4 Paper Printing Black and White or Color.



Blank document presets have predefined size, color mode, unit, orientation, positioning, and resolution settings. 


You can modify these settings before creating a document using the preset.


Presets are categorized into these sets:


    Photo
    Print
    Art & Illustration
    Web
    Mobile
    Film & Video


Access the New Document dialog 

1. Open Photoshop

2. Do one of the following keyboard shortcut to Access the New Document dialog

    (Mac) Cmd+N

    (Windows) Ctrl+N

3. or Select File -> New


Quickly access files and items that you've have recently accessed (Recents tab).


Save your own custom presets for reuse and quickly access them later (Saved tab).


Create documents using Blank Document Presets for multiple categories. Before opening the presets, you can modify their settings.


Create documents using presets

1. Select Print Category -> Select A4 preset  

Optionally, change the settings for the selected preset in the Preset Details pane on the right.

Leave Settings Default for now -> Click Create to creating a document using the preset.


2. or if opened Document already Right-click the name tab for an open document and choose New Document from the in-context menu.




Modify presets

Before opening a document using the preset, you can modify its settings in the right pane.

1. You can Specify a file name for the new document.

2. You can Specify Width and Height: Specify the size of the document. 

3. You can Specify unit from the pop-up menu of the document. 

4. You can Specify Orientation: Specify a page orientation for the document: Landscape or Portrait.

5. You can Select this option if you want your document to have artboards. Photoshop adds an artboard while creating the document.

6. You can Specify a color mode for the document. Bitmap, Grayscale, RGB Color, CMYK Color, Lab Color.

7. You can Specify the Resolution of image measured in pixels/inch or pixels/centimetre.

8. You can Specify a background color for the document.

9. Click Create to open a document with the preset settings.




Save your own presets

The Preset Details pane lets you modify an existing preset or specify fresh settings for a new one. 

To save your custom settings as a new preset, follow these steps:

1. After you're done specifying the settings, click the  download icon or save present button in the Preset Details pane.

2. Specify a name for the new preset.

3. Click Save Preset.

You can later access the new preset from the Saved tab of the New Document dialog.


Use the legacy File New experience

you can disable the latest New Document experience and revert to the File New experience that Photoshop provided by default in Photoshop versions 7.0

1. Select Edit -> Preferences -> General.

2. Select Use Legacy File New Interface. Click OK.
